## 2026-06-11 - Native Streamlit Tooltips for UX and A11y
**Learning:** Adding the `help` argument to Streamlit input widgets (`st.text_input`, `st.form_submit_button`, etc.) is a very lightweight and effective way to instantly provide native hover tooltips and generate underlying ARIA labels for screen readers without any custom CSS.
**Action:** Consistently leverage the `help` argument on all interactive form elements across Streamlit pages to boost accessibility and user context.
